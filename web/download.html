4D Building Blocks Download

You can download the application as Java Archive.
And then run it as an Application instead of as an Applet.
Thus additionally saving and loading configurations are enabled.
(Though not yet implemented ...)

Installation Instructions

  1. Put the downloaded file "@jar@" somewhere you like.
  2. Have the proper @jre_html@ installed.
  3. Either you are
    1. on windows: Right click on "@jar@" and choose "Open with > Java ... ".
    2. on any platform: Run "java -jar @jar@"
  4. Enjoy the game and give me feedback.
